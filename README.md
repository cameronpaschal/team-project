# Minimal Phone

Minimal Phone lets you pick which apps to block so you can stay on track. When a blocked app is opened, the user sees a message and gets redirected to the home screen. You can also pre-select apps for quick toggling of Focus Mode.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Android Studio or IntelliJ with an Android emulator
2. Open the project
3. Run it on either: An emulator / A physical Android device

Instructions for using the software:

1. Grant the required permissions: Accessibility, Usage Access, Overlay Permission
2. Tap Manage Focus to choose which apps to block
3. Use the home screen toggle to switch Focus Mode on/off
